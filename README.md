# README

# 1️⃣ 기술 스택
    - SpringBoot
    - JPA
    - MySQL
    - java 11
    - Spring Security + JWT

### 가계부 내역 등록[POST]
    http://localhost:8080/account/events

    {
      "email" : "kim@naver.com",
      "action" : "ADD",
      "money" : "100",
      "content" : "test content"
    }
    
---
    
    {
    	"result": "Success",
    	"type": "요청한 가계부 등록이 성공하였습니다",

        		"money": 100,
        		"content": "test content"
    	}
     }
